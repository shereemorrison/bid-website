# Dependencies

| Package | Role |
| --- | --- |
| **react** / **react-dom** | UI |
| **vite** | Dev server and production build |
| **tailwindcss** / **@tailwindcss/vite** | Styling |
| **gsap** + **ScrollTrigger** | Intro iris, hero copy, scroll reveals, parallax scrub |
| **lenis** | Smooth scroll (synced with GSAP ticker in `LenisProvider`) |
| **framer-motion** | Nav, scroll bar, merch card hover, pointer glow |
| **react-icons** | Nav and merch icons |

Motion timings live in `src/animations/timings.js`.
